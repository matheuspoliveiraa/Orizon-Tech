# 🌐 Orizon Tech - Site Institucional

Este projeto consiste no desenvolvimento de um **site institucional** para a empresa **Orizon Tech**, com o objetivo de apresentar a empresa, seus serviços e sua equipe de forma profissional.

## 🚀 Objetivo

Criar um site moderno e tecnológico que represente a identidade da empresa e apresente suas soluções voltadas para **estruturação, desenvolvimento e manutenção de sites**, principalmente no segmento de **E-commerce**.

## 🛠 Tecnologias Utilizadas

* HTML5
* CSS3
* Google Fonts
* GitHub

## 📄 Estrutura do Site

## 🎨 Design

O design do site segue um estilo **moderno e tecnológico**, utilizando cores escuras com destaque em tons de **verde**, alinhado com a identidade visual da empresa.
